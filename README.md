# Cryptogotchi
Adopt a cryptocurrency in this Tamagotchi style web game

## Usage
```bash
# Clone the repo.
$ git clone https://github.com/JamesFator/Cryptogotchi.git && cd Cryptogotchi
# Simply open the HMTL file.
$ open index.html
```
